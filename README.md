## GIT COMMANDS
```
# cd (change directory)
$ cd abcd
# to get into abcd directory
$ cd ..
# to come to previous director
$ cd ../..
# to come back to two prev directories (if exist)

#mkdir (make directory)
$ mkdir abcde
# to make new folder with name abcde
$ echo > xyz.abc
# to make a file named xyz with extension abc
example:
$ echo > index.html
# creating an html file named index
```
